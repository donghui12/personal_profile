# 🧰 常用工具与平台

## ⚙️ 开发与部署工具

- `Git`：版本控制
- `Docker` & `Kubernetes`：容器化及集群部署
- `Linux / macOS`：日常开发系统
- 编辑器：
  - `VSCode`：日常开发主力工具
  - `IntelliJ IDEA`：用于 Go 与多语言项目

## 🛠️ DevOps / CI/CD 工具

- `Jenkins`
- `GitHub Actions`
- 自研 DevOps 平台：参与或主导内部平台搭建与维护

## 📜 相关认证

- `CKA`：Certified Kubernetes Administrator
- `CKAD`：Certified Kubernetes Application Developer
